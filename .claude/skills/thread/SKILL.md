# /thread — X（Twitter）投稿ドラフト生成

## 概要
今日の開発成果・体のデータ・気づきから、X投稿のドラフトを3パターン生成する。
5分30秒以内で投稿できるレベルの短さが絶対条件。

## 手順

### 1. 今日のデータ収集
以下を自動で取得:
- 今日のデイリーノート: `/Users/og3939397/Documents/Obsidian Vault/zDOG/Daily/{今日}.md`
- 今日のコミット: `git log --oneline --since="today 00:00"`
- コミット数: `git log --oneline --since="today 00:00" | wc -l`

### 2. テーマ判定
`$ARGUMENTS` でテーマ指定があればそれに従う。なければデータから自動判定:
- コミット5本以上 → `dev`（開発報告）
- 体重データあり → `training` or `meal`
- メンタルスコアあり → `mental`
- それ以外 → `insight`

### 3. 投稿ドラフト生成（3パターン）

#### ルール（絶対遵守）
- **3-5行以内**（短い。これが最重要）
- **正直に書く**（体重増えても、メンタル低くても隠さない）
- **気づきを1つ入れる**（ただの記録じゃなく学びを）
- **ハッシュタグ 2-3個**
- **絵文字は最小限**（0-1個）
- 完璧な文章じゃなくていい。投稿すること自体が価値

#### パターンA: 数字ベース（Type 3: Number Attack）
```
MuscleMap v1.1
今日のコミット: {N}本
{最も重要な変更を1行で}

#個人開発 #MuscleMap
```

#### パターンB: 気づきベース（Type 5: Thought Memo）
```
{今日の開発/トレーニングで学んだこと1-2行}

{なぜそう思ったか1行}

#ビルドインパブリック #個人開発
```

#### パターンC: 進捗報告（Type 2: Training/Dev Log）
```
{具体的な成果を1行}
{数字で裏付け}

{次にやること1行}

#MuscleMap #SwiftUI
```

### 4. ハッシュタグ候補
テーマに応じて選択:
- 開発: `#個人開発` `#MuscleMap` `#SwiftUI` `#ビルドインパブリック` `#ClaudeCode`
- 筋トレ: `#筋トレ` `#減量` `#MuscleMap`
- メンタル: `#メンタルヘルス` `#システム思考`
- 食事: `#食事管理` `#減量`
- 気づき: `#システム思考` `#AI活用`

### 5. 出力
- 3パターンを番号付きで表示
- 「どれ使う？」と聞く（ユーザーが選んでコピペするだけの状態にする）
- 編集リクエストがあれば即座に修正
