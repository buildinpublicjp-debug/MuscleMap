# Design System Check

Read `docs/DESIGN_SYSTEM.md` and verify the current implementation against all 6 principles.

For the file(s) I'm about to modify:
1. Read the file(s)
2. Read docs/DESIGN_SYSTEM.md
3. Identify any violations of the 6 principles
4. List specific fixes needed
5. Do NOT make changes yet - just report findings

Principles to check:
- Principle 1: No dead whitespace
- Principle 2: Information priority reflected in size
- Principle 3: GIFs are the biggest weapon (identifiable size, dark card bg)
- Principle 4: Fill the screen (80%+ content)
- Principle 5: Consistency across screens
- Principle 6: Always verify with simulator screenshot
