# MuscleMap

筋肉の回復状態を可視化し、最適なトレーニングを導くiOSアプリ

**コンセプト:** 「筋肉の状態が見える。だから、迷わない。」

## 技術スタック
- iOS 17.0+ / SwiftUI / Swift Data
- RealityKit (3D) / SVG (2D)
- RevenueCat (課金)
